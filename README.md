# Night Owls Detector

Script prints devman users, which sent task from 24:00 till 06:00

# How to start

Python version 3.5 is required

Start on Linux:

```bash
pip install -r requirements.txt # alternatively try pip3

$ python3.5 seek_dev_nighters.py

```
